The Extraframes Mod is a modification to the game Minecraft that helps by adding additional performance enchancements through enhancements in its structure. Some enhancements include multithreaded chunkloading, multithreaded particle rendering, more efficient usage of the CPU and GPU cache, more efficient vertex buffer objects, more efficient methods associated with the sine and cosine caculations, faster texture application and registration, faster item, block, and entity registration allowing for improved load times, smart applications of particle culling, and an more updated and open source version of OpenGL which allows for community contributions towards the library for more performance improvements.
